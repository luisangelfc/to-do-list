# to-do-list
Una lista de tareas simple e interactiva creada con HTML, CSS y JavaScript. ¡Cookies habilitadas! Tus datos ahora se guardan de forma segura en tu navegador.

### Características

* Agregar nueva tarea
* Marcar tarea como completada / pendiente
* Eliminar una tarea individual
* Eliminar todas las tareas
* Actualizar nombre de usuario


### Tecnologías utilizadas:
* HTML
* CSS
* JavaScript
* SweetAlert2
